# party-shaking
## “聚会摇摇蛋”
**应用简介：**
    是一款Android手机游戏App，旨在聚会场景下，提供随机选择互动小游戏和惩罚的功能，为聚会活跃气氛专用！
    
**开发团队：**    
    由中国，成都，电子科技大学的戴嘉乐，周程健，陈南松于2013年开发
